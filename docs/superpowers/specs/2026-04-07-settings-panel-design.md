# Settings Panel - Design Specification

## Overview
Slide-out settings panel with terminal settings, tunnel controls, help, and about sections.

## Components

### 1. Settings Panel (Slide-out)
- Toggle via gear icon [⚙] in top bar
- Close via X button or click outside
- Sections: Terminal, Tunnel, Help, About

### 2. Terminal Settings
- Font Size: 12-20px slider
- Theme: Dark (default), Light, High Contrast
- Scrollback: 500, 1000, 5000, 10000 lines

### 3. Tunnel Controls
- Status indicator (Connected/Disconnected)
- Start cloudflared button
- URL display and copy button
- Stop button

### 4. Help Section
- Keyboard shortcuts list
- Supported commands
- Troubleshooting tips

### 5. About Section
- Version info
- Link to full README
