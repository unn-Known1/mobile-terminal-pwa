# Mobile Terminal

A fully functional terminal environment for mobile devices. SSH into your servers, run scripts, and work on the go — all from your phone or tablet.

## ✨ Features

- Optimized terminal UI for touch screens
- Gesture support: swipe, pinch-to-zoom, long-press
- SSH client with key management built in
- Works on Android, iOS, and iPadOS

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/unn-known1/mobile-terminal.git
cd mobile-terminal

# Install dependencies
pip install -r requirements.txt

# Run the app
python app.py

# Connect to your server via SSH within the app
```

## 🏗️ Stack

- **Frontend:** React Native (or Flutter)
- **Terminal:** xterm.js compatible
- **Backend:** Python
- **Platform:** Android & iOS

## 💡 Use Cases

- Admin on the go — manage servers from your phone
- DevOps engineers needing mobile terminal access
- Students and learners practicing CLI on mobile
